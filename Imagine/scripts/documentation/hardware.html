A. Exposure Trigger Input

A signal for External Exp. Start or External Exp. Ctrl trigger mode is connected.

B. Acquire Enable Input

A signal for Acquire Mode or Sequence Trigger Mode is connected.

C. Status Busy

A signal indicating busy status is given at the status busy output. Once an acceptable trigger edge is received, busy will go on status high. As soon as busy goes low again, a new trigger edge is accepted.

D. Status Expose

A signal indicating exposure status is given at the status output. Status Expos indicates the actual exposure window for one frame.

Sensor ROI VS Soft ROI




The senor ROI is limited with some rules to increase capturing speed. For the OCPI-2 camera, H start and H stop has step size of 160 pixels. And V start and V stop should be symmetric across the center horizontal line. Soft ROI is more free to set but the capturing speed is limited as the same speed of the smallest sensor ROI which can include the soft ROI.
